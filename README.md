# Furniture Project

## Description
An online furniture store application where users can browse products, add them to the cart, place orders, and make payments. Admin panel allows managing products, categories, and orders.

## Tech Stack
- Java
- Spring Boot
- MySQL
- Thymeleaf (frontend)
- Maven
